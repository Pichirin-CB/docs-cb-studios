# Getting Started

[Home](/home.md) / Getting Started

Use this page as the first stop before installing or updating a CB Studios resource.

## Recommended Workflow

1. Find the exact product page in the sidebar or on [All Products](/products.md).
2. Read the product overview, requirements, dependencies, installation steps, and troubleshooting notes.
3. Confirm framework compatibility before installing on a production server.
4. Back up the current resource folder and configuration files before updating.
5. Test on a local or staging server whenever possible.
6. Use [Support](/support.md) only after collecting logs, reproduction steps, product version, and framework information.

<div class="docs-callout docs-callout--warning">
  <strong>Do not rename resource folders unless the product documentation says it is allowed.</strong>
  <p>Some resources validate their folder name for security or compatibility reasons.</p>
</div>
