# docs-cb-studios

Documentacion de CB Studios en Docsify.

## Requisitos

- Node.js 18+ (recomendado: Node.js 20)
- npm

## Configuracion local

1. Instalar dependencias:

```bash
npm install
```

2. Levantar Docsify en local:

```bash
npm run dev
```

3. Abrir en navegador:

```text
http://localhost:3000
```

## Notas

- Ya no dependes de tener `docsify-cli` instalado globalmente.
- Si alguna vez falla `npx` por Windows, verifica que exista esta carpeta:
  `C:\Users\<TU_USUARIO>\AppData\Roaming\npm`
