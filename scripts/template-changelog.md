# 🏷 Versioning

Recommended version format:

    MAJOR.MINOR.PATCH

Example:

-   **MAJOR** → big system rewrite or breaking changes
-   **MINOR** → new features or major improvements
-   **PATCH** → bug fixes and small adjustments

Example versions:

-   `1.0.0` Initial release
-   `1.1.0` New features added
-   `1.1.1` Bug fixes
