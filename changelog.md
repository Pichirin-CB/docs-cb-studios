# 🧾 CHANGELOG

## 🚀 | cb_deadzone-extract - `1.1.0` - 2026-03-12

### ✨ Highlights

- Removed dependency on `hate-bridge`.
- Script is now more independent and easier to install.

### 🔧 Core Changes

- Refactored core internal structure.
- Improved modular design for future expansion.
- Improved maintainability for future feature development.

### 📦 Compatibility

- Added support for additional inventory systems:
  - `qs_inventory`
  - `core_inventory`
  - `tgiann-inventory`
- Improved compatibility with different server environments.

### 🛠 Custom Integrations

- Added `custom.lua` integration hooks for inventory and notification systems.
- Developers can now extend support without modifying core files.

### 🔔 Systems & Features

- Added support for custom notification systems through integration hooks.

### 🌍 Localization

- Added new languages:
  - French
  - Portuguese

### 🧩 General Improvements

- Better flexibility for server developers.
- Easier customization and maintenance.

### 🗂️ Files Modified

- `server/inventory/custom.lua`
- `client/inventory/custom.lua`
- `client/notifications/custom.lua`
- `server/inventory/init.lua`
- `client/inventory/init.lua`
- `client/notifications/init.lua`
- `locales/fr.lua`
- `locales/pt.lua`
