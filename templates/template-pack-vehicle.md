# Installation

### 1 Download

Download the vehicle pack and extract it.

### 2 Place in resources

    resources/[your_folder]/cb_vehicle_pack

### 3 Add to server.cfg

    ensure cb_vehicle_pack

### 4 Restart server

    restart cb_vehicle_pack

# Common Issues

### Vehicle does not spawn

-   Verify resource name
-   Ensure pack is started in server.cfg
-   Check for conflicting vehicle names

### Missing textures

-   Verify all files are included
-   Clear FiveM cache

### Vehicle has no sound

-   Verify audio configuration
-   Check for conflicting sound resources

------------------------------------------------------------------------

# Updating the Pack

1 Stop the resource 2 Backup the current version 3 Replace with the new
