# Resource Structure

Example structure of the mapping resource:

cb_example_mlo/

│ ├─ fxmanifest.lua ├─ README.md │ ├─ stream/ │ ├─ interior files (.ydr
/ .ybn / .ymap) │ └─ meta/ ├─ manifest or meta files

Note: Folder names may vary depending on the map.

# Installation

### 1 Download

Download the MLO resource and extract it.

### 2 Place in resources folder

resources/\[your_folder]/cb_example_mlo

### 3 Add to server.cfg

ensure cb_example_mlo

### 4 Restart server

restart cb_example_mlo

### 5 Verify

Join the server and go to the location of the map to confirm it loads
correctly.

# Common Issues

Map not loading

• Verify resource name • Ensure the resource is started • Check server
console for errors

Textures missing

• Clear FiveM cache • Verify all files are present • Check stream folder

Interior collision issues

• Ensure no conflicting map replaces the same area

------------------------------------------------------------------------

# Updating the Map

1 Stop the resource 2 Backup the current version 3 Replace files with
the updated version 4 Restart the resource 5 Verify the map loads
correctly
